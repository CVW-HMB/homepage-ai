# vince-welke.com

Personal portfolio site with an AI assistant that answers questions about my experience.

**Live:** [vince-welke.com](https://vince-welke.com)

## Tech Stack

- **Framework:** Next.js 15, TypeScript, Tailwind CSS
- **AI:** OpenAI API (gpt-4o-mini)
- **Hosting:** Vercel
- **CI/CD:** GitHub Actions

## Features

- AI assistant trained on my resume and project history
- Rate limiting and bot detection to prevent abuse
- Fully responsive design

## Run Locally

```bash
npm install
npm run dev
```

Requires `.env.local` with `OPENAI_API_KEY`.
